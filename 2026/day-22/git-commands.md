# Git Commands Reference

## Setup & Config

- git --version  
Checks installed Git version  
Example: git --version

- git config --global user.name  
Sets global username  
Example: git config --global user.name "Namit"

- git config --global user.email  
Sets global email  
Example: git config --global user.email "namit.ranjan@gmail.com"

---

## Basic Workflow

- git init  
Initializes a new Git repository  
Example: git init

- git status  
Shows working directory status  

- git add <file>  
Stages changes  

- git commit -m "message"  
Commits staged changes  

---

## Viewing Changes

- git diff  
Shows unstaged changes  

- git log  
Shows commit history  

- git log --oneline
